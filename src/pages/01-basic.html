1.1 - What is HTML?

HTML stand for HyperText Markup Language.it is the standared language use and structure content on the web. HTML tells the browser what each piece of content it - a heanding, paragrapha, image, limk, etc. - using tags.

<!-- This is an HTML comment -->

Hello, World!

Output

Hello, World!

1.2 - The DOCTYPE declaration

Every HTML file must start with <!DOCTYPE html>.This tells the browser that the page the uses the modern HTML5 standard. Without it, the browser may render the page incorrectly


            <!DOCTYPE html>
            <html lang="en">
            <head>
            <title>My First Page</title>
            </head>
            <body>
            <p>Page content goes here.</p>
            </body>
            </html>

1.3 - HTML Page Structure

An HTML document has a specific structure. The <html>tag wraps everything. inside it are two main sections: <head>(invisible metadata) and <body>(visible content) 


            <html>         ← Root element, wraps all
            <head>       ← Metadata: title, charset, links
            ...
            </head>
            <body>       ← Visible page content
            ...
            </body>
            </html>

Output

This text is inside the <body>— it's what users see.

1.4- HTML Element & Tags

Anelementconsists of an opening tag, content, and a closing tag. Some are self-closing and don't need a closing tag (like <br> or <img>.)


        <p>This is a paragraph element.</p>

       <br>        ← Self-closing: just a line break
       <img src="photo.jpg" alt="A photo">  ← Self-closing

Output

This is a paragraph element.

(br adds a line break · img renders an image)

1.5 - Attributes

Attributesgive extra information to element. They are written inside the opening tag as name="value" pairs. Common attributes include id, class, src, href, and alt,


        <a href="https://example.com" target="_blank">Visit Site</a>

        <img src="logo.png" alt="Company logo" width="200">

        <p id="intro" class="highlight">Welcome!</p>

Output

Welcome! (class="highlight" would apply CSS styles)

1.6 - Nesting Elements

Elements can be placed inside other element - this callednesting.Always close inner element before outer ones. Imporper nesting causes browser rendering issuse.


✅ Correct nesting:
<p>This is <strong>bold</strong> text.</p>

❌ Wrong nesting:
<p>This is <strong>bold</p></strong>

Output

This is boldtext.
